# 도서관 프로젝트 포트폴리오 페이지 설계

**작성일:** 2026-05-21
**프로젝트:** portfolio-library
**목적:** 완료된 도서관 관련 프로젝트 소개 및 개인 기록 아카이빙

---

## 1. 개요

빈윤미(사서)가 바이브코딩으로 만든 도서관 관련 6개 프로젝트를 소개하는 단일 HTML 포트폴리오 페이지.
빌드 없이 브라우저에서 바로 열리며, 나중에 프로젝트를 추가할 때 HTML 블록만 복붙하면 되도록 설계.

---

## 2. 기술 스택

- **순수 HTML + CSS + JavaScript** (빌드 도구 없음)
- 외부 의존성: Google Fonts (Noto Serif KR, Noto Sans KR) — CDN
- 단일 파일: `portfolio-library/index.html`

## 4. 디자인 가이드

### 컬러 팔레트

| 역할 | 색상 | 코드 |
|------|------|------|
| 페이지 배경 | 크림/베이지 | `#FDF6F0` |
| 헤더 배경 | 짙은 우드 브라운 | `#5C3D2E` |
| 강조색 | 테라코타 | `#C9622F` |
| 카드 배경 | 흰색 | `#FFFFFF` |
| 본문 텍스트 | 다크 그레이 | `#333333` |
| 보조 텍스트 | 미디엄 그레이 | `#777777` |
| 카드 테두리 | 연한 베이지 | `#EDE0D4` |

### 타이포그래피

| 용도 | 폰트 | 굵기/크기 |
|------|------|----------|
| 프로젝트명/섹션 제목 | Noto Serif KR | Bold, 18~22px |
| 본문/설명 | Noto Sans KR | Regular, 14~15px |
| 태그 | Noto Sans KR | Medium, 11~12px |

### 레이아웃
- 최대 너비: 960px, 중앙 정렬
- 카드 그리드: PC 2열, 모바일 1열
- 카드: 흰색 배경 + 연한 그림자 + 부드러운 모서리(12px)

## 5. 프로젝트 카드 구성

각 카드에 포함되는 요소:
1. 이모지 + 프로젝트명
2. 한 줄 설명
3. 주요 기능 2~3개 (bullet)
4. 기술 스택 태그
5. 배포 링크 버튼 (없으면 "로컬 운영 중" 표시)

---

## 6. 프로젝트 목록 (포트폴리오 표시 순서)

### 1. MY LAB
- **폴더:** `library app/app`
- **설명:** 학생들이 스마트폰으로 도서를 대출/반납하고 독서활동을 관리하는 학교 도서관 앱
- **주요 기능:**
  - 학생 바코드로 도서 대출/반납 처리
  - 도서 검색 및 감상문 작성
  - 반납일 푸시 알림
- **스택:** Flutter, Firebase (Firestore, FCM, Auth)
- **배포:** 앱 (로컬 운영 중)

### 2. 희망도서 신청
- **폴더:** `library-wish`
- **설명:** 학생·교직원이 원하는 책을 검색하고 신청하는 희망도서 관리 웹앱
- **주요 기능:**
  - 알라딘 API로 도서 검색 및 신청
  - 신청 내역 조회
  - 관리자 엑셀 다운로드
- **스택:** React, TypeScript, Google Apps Script
- **배포:** Vercel (GitHub: drm815/library-web-app)

### 3. LibraBook
- **폴더:** `timetable_lib`
- **설명:** 선생님·학생이 도서관 수업 시간과 방과후 좌석을 예약하는 시스템
- **주요 기능:**
  - 교시별 수업 예약 및 충돌 협의
  - 학생 방과후 좌석 배치도 예약
  - 관리자 월별 타임테이블 설정
- **스택:** Next.js, TypeScript, Supabase, Tailwind CSS
- **배포:** Vercel

### 4. 해밀도서관 포털
- **폴더:** `haemil-library`
- **설명:** 해밀도서관의 모든 서비스를 한 곳에서 이용하는 통합 포털
- **주요 기능:**
  - 도서관 서비스 바로가기 카드
  - QR코드 생성 및 인쇄 페이지
  - 개인정보처리방침 안내
- **스택:** Next.js, TypeScript, Tailwind CSS
- **배포:** Vercel

### 5. 복본도서 폐기 관리
- **폴더:** `book-check`
- **설명:** 10권 이상 보유 복본도서를 교사에게 공유해 수업 활용 계획을 수집하고 폐기 대상을 선별하는 웹앱
- **주요 기능:**
  - 관리자 엑셀 업로드 → 교사 공유 링크 생성
  - 교사 활용 계획 체크 및 제출
  - 폐기 대상 목록 엑셀 다운로드
- **스택:** Next.js, TypeScript, Supabase, Tailwind CSS
- **배포:** Vercel

### 6. 책정리 도우미
- **폴더:** `booksort`
- **설명:** 도서부 학생이 바코드를 스캔하면 해당 책의 서가 위치(앞뒤 3권)를 시각화해주는 모바일 웹앱
- **주요 기능:**
  - 바코드 스캔으로 서가 위치 즉시 확인
  - KDC 청구기호 기준 앞뒤 3권 선반 시각화
  - 구글 시트 실시간 연동 (30분 캐시)
- **스택:** React, TypeScript, Vite, Tailwind CSS, Google Sheets
- **배포:** Vercel

## 7. 페이지 구조

```
index.html
├── <head> — 폰트, 스타일 인라인
├── <header> — 프로필 섹션 (브라운 배경)
├── <main>
│   ├── <section class="projects"> — 프로젝트 그리드
│   │   └── 카드 × 6
└── <footer> — 카피라이트
```

## 9. 확장성 고려
